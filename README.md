# Mechanical-Design-Projects Solidworks

## Jet Engine Assembly Design

Designed a complete 3D jet engine assembly in SolidWorks using parametric modeling techniques.\
Created individual components and assembled them using advanced mates.\
Performed mass, volume, center of mass, and weight analysis using SolidWorks Mass Properties.\
Developed exploded view and assembly animation to demonstrate working principles.\
Prepared detailed engineering drawings with dimensions and annotations.

Tools: SolidWorks, Assembly, Motion Study, Drawing, Mass Properties\

Performed mass, volume, and center of gravity analysis.\
